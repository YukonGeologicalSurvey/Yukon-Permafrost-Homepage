Downloads

The data compiled and managed in the Yukon Permafrost Database is available for download as CSV files. We publish these datasets to an FTP site via weekly scripts.

  • Geotechical Borehole Dataset (2MB)
  • Temperature Sensor Dataset (160MB)

The database will be an evolving product that we’ll update periodically as new data sources become available.

For more information, please contact ygs-surficial@yukon.ca.
