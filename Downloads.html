Downloads

The data compiled and managed in the Yukon Permafrost Database are available for download in a variety of formats. We publish these datasets to an FTP site via weekly scripts. The database is an evolving product that we will update as new data sources become available.

Temperature Sensor Dataset Entire dataset CSV ~170MB
